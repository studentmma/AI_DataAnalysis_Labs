# Лабораторная работа №1: Исследовательский анализ данных (EDA)

**Дисциплина:** Анализ данных и искусственный интеллект
**Среда:** Python 3.10, Jupyter Notebook (Google Colab)

## Описание проекта
В данной работе проведен первичный анализ данных о характеристиках автомобилей. 
Цель работы — очистка данных от пропусков и выбросов, создание новых признаков 
и поиск зависимостей между техническими параметрами.

## Ключевые этапы:
* Выполнена визуализация пропусков с помощью heatmap.
* Обработаны выбросы в ценовых категориях.
* Реализована функция для автоматической очистки данных (заполнение медианой и модой).
* Создано 3 новых синтетических признака (удельная мощность, средний расход, ценовой диапазон).
* Построена матрица корреляции признаков.

## Использованные библиотеки:
Pandas, NumPy, Matplotlib, Seaborn

## Лабораторная работа №2: Подготовка данных

**Цель:** Очистка и преобразование признаков для машинного обучения.

### Выполненные задачи:
1. **Кодирование признаков:**
   - `Label Encoding` для порядковых данных (цилиндры).
   - `One-Hot Encoding` для категориальных данных (тип кузова).
2. **Масштабирование:** Использован `MinMaxScaler` для приведения числовых признаков к диапазону [0, 1].
3. **Feature Engineering:** Созданы новые признаки `HP_per_EngineSize` и `Weight_to_Wheelbase`.
4. **Очистка:** Удалены нерелевантные текстовые признаки (`Model`, `Manufacturer`), которые не несут пользы для обучения модели.

**Результат:** Подготовлен финальный датасет, полностью состоящий из числовых данных.
